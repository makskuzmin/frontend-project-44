Install: 
```
git clone git@github.com:makskuzmin/frontend-project-44.git
cd ./frontend-project-44
sudo npm link
```

Minimum requirements:
- install npm
- install Node.js 

Launching games:
Enter the name of the game in the console:
```
brain-even
brain-calc
brain-gcd
brain-progression
brain-prime
```
